# Python Test Scripts

These scripts run various benchmarks.
They are grouped as follows:

| Prefix     | Purpose                                                                                        |
| ---------- | ---------------------------------------------------------------------------------------------- |
| aetherling | Compare the hardware generated by our compiler vs Aetherling.                                  |
| ablation   | Ablation study (see what happens when optimization passes are turned off)                      |
| dse        | Design space exploration similar to Aetherling, but for benchmarks Aetherling doesn't support. |
